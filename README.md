# 🎺 練習コンパニオン

楽器練習の「気づき」を、試して、検証して、定着させる。
練習の上達サイクル（PDCA）を1画面で回すための、無料のWebアプリです。

## なぜ作ったか

練習中の発見は、その日のうちに忘れます。メモに書いても、メモは見返されません。
そして「あれもこれも試そう」とすると、結局どれも検証されずに終わります。

このアプリは、その2つの問題だけを解決します。

- **気づきは一箇所に貯まる**：練習後に書いた「次回試したいこと」が、自動で「試したいリスト」に入る
- **今日試すのは最大2個**：意図的な制限です。仮説は絞るから検証できる

## 使い方（1練習=1サイクル）

1. **練習前**：コンディションを記録し、「ためす」から今日試す仮説を1〜2個選ぶ
2. **練習中**：選んだ仮説を意識して吹く（弾く）
3. **練習後**：結果を記録する——✅定着 ／ 📌継続検証 ／ ❌否定
4. 「次回試したいこと」を書けば、次のサイクルの種が自動で仕込まれる

検証済みの発見は「知識」タブに貯まり、フラッシュカードのように見返せます。
あなただけの奏法知識ベースが育っていきます。

## プライバシー

**練習データはあなたのブラウザの中にだけ保存されます。**
サーバーには何も送信されません。アカウント登録もありません。

その代わり、機種変更やブラウザのデータ消去でデータは消えます。
「知識」タブの**バックアップ（JSON書き出し）**をときどき使ってください。

## 技術

- 依存ライブラリゼロの静的Webアプリ（HTML + Vanilla JS）
- PWA対応：スマホの「ホーム画面に追加」でアプリのように使えます。オフラインでも動きます
- データはlocalStorage。エクスポート／インポートで移行可能

### 開発

```bash
# テスト（Node.jsのみ必要）
bash run_tests.sh

# ローカルで動かす
python3 -m http.server 8080
# → http://localhost:8080
```
